## Installation

Clone down this repository. You will need node and yarn installed globally on your machine.

Installation :

```bash
yarn
```

To Start Server :

```bash
yarn dev
```

To Visit App :

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To Run Test Suite :

```bash
yarn test
```
